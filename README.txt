Wat staat er op de CD?

-Arduino code: bevat de code die draait op de Arduino module 
-Authenticatieserver: bevat de belangrijkste files van de authenticatieserver. 
-Autorisatieserver: bevat de belangrijkste files van de autorisatieserver. 
-basic-android: bevat het Cordova project van de basic versie van het prototype.
-code-push-server: bevat ALLE files van de lokale kopie van de CodePush cloud service.
-expert-android: bevat het Cordova project van de expert versie van het prototype.
-Scriptie: bevat pdf-files van de scriptie, de poster en het wetenschappelijk artikel.
-config: configuratiebestand dat de IP-adressen van de servers bevat.
